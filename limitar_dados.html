Limitar o tratamento de dados

Voltar para gestão de dados

Ao limitar o tratamento de um dado, nós deixaremos de utilizá-lo em nossos sistemas no contexto que você especificar. Limitar o tratamento de certos dados fará com que algumas usabilidades do nosso sistema não estejam disponíveis.
A limitação de quaisquer dados que não estão incluídos abaixo irá impossibilitá-lo de usar nosso serviço, como descrito na nossa Política de Privacidade. Caso queira, você pode Excluir todos os seus dados, o que acarretará na exclusão da sua conta.

Selecione os dados pessoais e o tipo de limitação que você quer aplicar a cada um dele:

  • Não utilizar meu email para divulgação

Limitar uso do Endereço

  • Não utilizar meu endereço para comunicação

Limitar uso do Telefone de Contato

  • Não repassar meu número de telefone para terceiros
  • Não utilizar meu telefone de contato para divulgação
